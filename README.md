# Esse repositório foi criado para disponibilizar programas em Python para solução de problemas de otimização.

# Os programas foram desenvolvidos como parte da disciplina ENG-493 Otimização de Sistemas Agro-Industriais ministrada para alunos de Engenharia Agrícola e Ambiental da Universidade Federal de Viçosa.

# Os programas disponibilizados foram desenvolvidos pelos professores Daniel Marçal de Queiroz e Domingos Sárvio Magalhães Valente.

# O respositório está dividido em capítulos, a idéia é que cada capítulo seja lecionado a cada semana. Os programas iniciais não tratam diretamente sobre otimização, mas sim de uma breve revisão sobre Python.

# Para rodar esses programas sugerimos que o usuário utilize a distribuição Anaconda do Python, com a versão Python 3.6 ou mais recente. Isso porque nos problemas envolvendo otimização será necessário ter as bibliotecas Numpy e Scipy instaladas. A distribuição do Anaconda já tem essas bibliotecas disponibilizadas. Uma outra biblioteca é utilizada  nos programas é a biblioteca de programação linear PulP.

# Os programas foram desenvolvidos por nós usando o conhecimento que vimos adquirindo com o passar do tempo. Caso seja encontrado algum bug (a gente não espera que não tenha), por favor, entre em contato conosco pelos email dmqueiroz@gmail.com.

# This repository was created to make available Python programs developed for solution of optimization problems.

This is beacuse in the problems involving optimization it is necessary to have the Numpy and Scipy libraries installed. And Anaconda distribution includes these librarires. One of the last chapters will make use of PulP a linear programming library.
